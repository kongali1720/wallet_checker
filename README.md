# 💸 Ethereum Wallet Generator + Saldo Checker

Script Python ini:
- Membuat wallet Ethereum (private key + address)
- Mengecek saldo wallet menggunakan Etherscan API

---

## 🛠 Cara Menjalankan

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
---
Dapatkan API Key gratis dari:
👉 https://etherscan.io/myapikey

Masukkan API key kamu ke dalam wallet_checker.py (ganti bagian ETHERSCAN_API_KEY)

---

2. Jalankan script sekarang:
   ```bash
   python wallet_checker.py
